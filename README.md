#### What I've built

**[Bench Resource Optimizer](https://github.com/RavSinghChandan/01-rag-chatbot)** — Enterprise AI HR platform. Maps bench employees to open roles using 5-layer Hybrid RAG (FAISS + BM25 + HyDE + CRAG + cross-encoder rerank), L1+L2 semantic cache, SSE streaming, circuit breaker, and LLM-as-judge guardrails. 502 tests. SonarQube Quality Gate PASSED.

**[RunbookAI](https://github.com/RavSinghChandan/04-document-qa)** — IT incident response engine with zero vectors, zero embeddings, zero hallucinated commands. Every `kubectl` command pulled verbatim from SQL. NetworkX DAG for step ordering + parallel execution detection. 141 tests.

**[AstroIntel 360°](https://github.com/RavSinghChandan/ai-engineer)** — 18+ LangGraph agents generating personalised reports across 5 spiritual domains in 23 Indian languages. Hybrid RAG + multi-query expansion. JWT + API key dual auth. 415 tests.

**[Agentic Growth OS](https://github.com/RavSinghChandan/03-ai-agent)** — 5-node LangGraph marketing pipeline with a learning engine that stores campaign performance (CTR, conversion, ROI) and uses it to improve the next run automatically.

**[Universal Agent](https://github.com/RavSinghChandan/03-ai-agent)** — Plug-and-play agent platform. Any app plugs in an AI agent via a YAML config file. Per-agent lock/unlock for token protection. Model-agnostic.
